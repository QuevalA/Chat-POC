# Chat POC
Real time chat proof of concept.

## Technologies
- Node.js
- Express.js
- Nodemon
- Socket.io
- Angular

## Instructions
### On the root level
- npm install
- npm run serve (uses port 3000 by default)

### On a 2nd terminal in the "client" folder
- npm install
- npm run build
- npm run start (uses port 4200 by default)
